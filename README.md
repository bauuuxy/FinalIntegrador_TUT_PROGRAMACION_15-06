## Descripción
Este proyecto es nuestro Trabajo Final Integrador, en el cual reunimos y resolvimos las actividades prácticas pedidas durante la cursada. El repositorio incluye los archivos desarrollados en PseInt (con ejercicios de lógica, texto y cálculos matemáticos) y en Python (para el armado de la red de 8 nodos y el de promedio).

Proceso de trabajo y pruebas:
Fuimos programando cada ejercicio asegurándonos de cumplir estrictamente con lo solicitado en las consignas. Al momento de pulir el código, aprovechamos para sumar mejoras más creativas en el diseño y la interacción de los programas, cuidando siempre de no alterar lo que pedía la actividad. Finalmente, probamos cada script con distintos datos para verificar que todo funcione de forma correcta y sin errores.
