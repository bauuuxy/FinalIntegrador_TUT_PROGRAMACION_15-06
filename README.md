## Descripción
Este proyecto es nuestro Trabajo Final Integrador, en el cual reunimos y resolvimos las actividades prácticas pedidas durante la cursada. El repositorio incluye scripts desarrollados tanto en PseInt (con ejercicios de lógica, texto y cálculos matemáticos) como en Python (para el armado de la red de 8 nodos).

Proceso de trabajo y pruebas:
Fuimos programando cada ejercicio asegurándonos de cumplir con todo lo solicitado en las consignas. Una vez listos, nos dedicamos a pulir el código para dejarlo limpio, optimizar las variables y corregir detalles. Finalmente, probamos cada programa con distintos datos de entrada para verificar que todo responda de forma correcta y funcione sin errores antes de la entrega.
